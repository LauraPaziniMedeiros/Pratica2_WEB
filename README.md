# Poções e Soluções - Web Service

Este projeto é um Web Service para a loja "Poções e Soluções" de Anna Innabelle Merigold, permitindo o cadastro, listagem e remoção de produtos, além de exibir uma vitrine para os clientes.

## Pré-requisitos
* Node.js instalado.

## Configuração e Execução
1. Clone o repositório e navegue até a pasta do projeto.
2. Inicialize o projeto e instale as dependências executando:
   ```bash
   npm install express sequelize sqlite3 cors
3. Para executar:
   ```bash
   npm start
